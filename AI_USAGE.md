# AI Usage Log

## Tools Used

- ChatGPT

## Purpose of AI Usage

AI was used as a development assistant throughout the project to help with:

- project planning
- Flask route scaffolding
- SQL query structure
- debugging Python and MySQL issues
- refining HTML templates and CSS organization
- improving documentation and setup instructions

AI was not used blindly. All generated output was reviewed, tested, modified, and integrated manually.

---

## Key Areas Where AI Helped

### 1. Project Structure and Planning

AI helped define an incremental milestone-based plan for the project, including:

- initial repository setup
- Flask application structure
- MySQL schema design
- staged CRUD implementation
- finalization and demo preparation

### 2. Database Schema and SQL Design

AI assisted in drafting:

- the four-table relational schema
- foreign key relationships
- JSON storage for job skills
- seed data for development and testing

### 3. Backend Development

AI helped scaffold and refine:

- database connection helpers
- CRUD data-access functions
- Flask route handlers
- validation logic
- job match feature calculation

### 4. Frontend and Templates

AI helped organize:

- reusable base template structure
- CRUD page layouts
- navigation flow
- styling improvements for readability and consistency

### 5. Debugging

AI was used to diagnose and fix issues such as:

- missing Python dependencies
- environment setup problems
- Git branch and upstream issues
- Python type annotation compatibility issues
- template rendering issues
- incorrect working directory and schema loading problems

## What I Learned About Using AI Effectively

Using AI was most effective when:

- prompts were specific and grounded in the current codebase
- output was tested immediately after generation
- debugging was done incrementally instead of changing too much at once
- AI was used to accelerate structure and troubleshooting, not to replace understanding

I learned that AI is most useful as a collaborative development tool when combined with careful review, local testing, and manual refinement.
